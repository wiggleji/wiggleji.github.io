## 환영합니다!

이것은 **Gatsby**로 만들어 GitHub Pages에 배포한 제 첫 번째 블로그 글입니다.

소프트웨어 개발에 대한 생각을 공유하고, 배운 것들을 기록하기 위해 이 블로그를 만들었습니다.

## 블로그 기능

이 블로그에 구현한 기능들입니다:

- 쉬운 탐색을 위한 태그와 카테고리
- 스크롤을 따라오는 목차
- Solarized Light 테마의 코드 하이라이팅
- Mermaid 다이어그램 지원
- 이중 언어 지원 (영어 및 한국어)

## 코드 예시

코드 하이라이팅을 테스트하기 위한 간단한 Python 예시입니다:

```python
def fibonacci(n: int) -> list[int]:
    """피보나치 수열을 n개 항까지 생성합니다."""
    if n <= 0:
        return []

    sequence = [0, 1]
    while len(sequence) < n:
        sequence.append(sequence[-1] + sequence[-2])

    return sequence[:n]

print(fibonacci(10))
```
