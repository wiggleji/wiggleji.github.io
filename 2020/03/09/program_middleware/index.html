Middleware

아래는 Wikipedia에서 Middleware에 대한 정의이다.

Middleware is computer software that provides services to software applications beyond those available from the operating system.

우리가 OS 위에서 소프트웨어를 실행하면 미들웨어는 해당 소프트웨어와 OS 사이에서 여러 서비스를 제공한다는 말인데

Wikipedia 이미지 를 보면 Linux 환경에서 게임을 실행했을 때, 일어나는 여러 이벤트 구조를 HW/SW 로 나누어진 걸 볼 수 있다. 여기서 미들웨어는 Linux kernel과 Game 사이에서 일어나는 Input, Output에 대한 처리를 해준다.

우리가 흔히 접하는 미들웨어는 Database middleware, Application server middleware, Message-oriented middleware, Web middleware 등이 있다.

Python 백엔드 공부를 하며 만난 Gunicorn과 같은 WSGI 또한 미들웨어이다.
